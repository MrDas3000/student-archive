# 🎓 Student Academic Archive System

A full-stack academic management platform designed to help students and administrators manage academic resources efficiently.

This system provides centralized access to **courses, notes, attendance, question papers, and academic logs** — making academic data organized, searchable, and accessible from anywhere.

---

## 🚀 Live Features

✅ Student dashboard with attendance tracking
✅ Course archive with search functionality
✅ Question paper repository
✅ Notes upload & download system
✅ Admin panel for full content management
✅ User authentication & role-based access
✅ Activity logging system
✅ Secure file handling & downloads

---

## 🧩 Problem It Solves

Students often struggle to find:

* Previous year question papers
* Organized course materials
* Class notes & academic resources
* Attendance records

This system provides a **single digital academic archive**.
